# 🎧 AudioPDF Reader

AudioPDF Reader is a Python-based application that converts text from PDF files into speech using a text-to-speech engine. It allows users to listen to PDF content instead of reading it, making it useful for students and accessibility purposes.

---

## 🚀 Features
- Select any PDF file from your system
- Extract text from all pages
- Convert text to speech using pyttsx3
- Simple and lightweight application

---

## 🛠️ Technologies Used
- Python
- PyPDF2
- pyttsx3
- Tkinter (file dialog)

---

## 📦 Installation

Install the required libraries:

```bash
pip install pyttsx3 PyPDF2
```

How to Run
Open the project in VS Code
Run the Python file:
python pdf_reader.py
